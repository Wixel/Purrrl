Purrrl
=============

Purrrl is a simple way to share files publically using your command line.

## Installation

```shell
gem install purrrl
````

## Example Usage

```shell
purrrl login you@you.com yourpassword
````

```shell
purrrl upload filepath
````

```shell
purrrl files
````
